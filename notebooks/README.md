# notebooks

Shared Colab notebooks for running experiments. All model training should be done here (not locally) to ensure everyone uses the same GPU environment and library versions.

Example notebooks:
- `01_data_exploration.ipynb` — visualise generated images, sentence distributions, dataset statistics
- `02_baseline_training.ipynb` — train and evaluate the baseline CNN
- `03_experiments.ipynb` — analytic axis experiments (text representation comparison, MLP vs CNN)

To use: open in Google Colab, mount Google Drive, and point data paths to the shared Drive folder.
